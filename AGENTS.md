# Conventions

- Commit often with short, meaningful messages.
- Keep code short and simple. Prefer the simpler implementation over extra features.
- Use Ruby stdlib only — no gems.
- External tools (`du`, `find`) are acceptable for performance.
- Chart.js is vendored in `vendor/` and version-locked.
- `index.html` is generated by `generate.rb` from `index.erb` — do not edit manually.
- `data.yml` is generated by `generate.rb` — do not edit manually.
- `style.css` and `script.js` are the static assets — edit for visual/layout changes.
- `index.erb` is the HTML template — edit for layout changes.
- `config.yml` is the config file — edit to change monitored folders.
